## ![use-case](../images/use-case-14.svg) Use Cases for CP4S/SOAR
<br/>

##### SOAR

Formerly known as Resilient, SOAR, is a core CP4S component that delivers security orchestration, automation, and response.

**Assets:**

* <a href="https://ibm.seismic.com/Link/Content/DCY-QNRfUcwEyqCViSDLciww" target='_blank' rel='noreferrer noopener'>IBM Cloud Pak for Security 301 SOAR Presentation </a>

* <a href="https://www.ibm.com/products/cloud-pak-for-security/soar" target='_blank' rel='noreferrer noopener'>IBM Security SOAR external product page </a>

* <a href="https://www.ibm.com/docs/en/cloud-paks/cp-security/1.8?topic=case-management-orchestration-automation" target='_blank' rel='noreferrer noopener'>Case Management and Orchestration & Automation documentation</a>

* <a href="https://ibm.seismic.com/Link/Content/DCDH6waBAfRkeVnaQevXibAA" target='_blank' rel='noreferrer noopener'>IBM Security SOAR Client Presentation</a>

<br/>

##### Incident response

Guide the incident response process with knowledge to be more efficient and effective.

**Assets:**

* <a href="https://www.ibm.com/products/cloud-pak-for-security/soar" target='_blank' rel='noreferrer noopener'>IBM Security SOAR external product page </a>

* <a href="https://www.ibm.com/docs/en/cloud-paks/cp-security/1.8?topic=case-management-orchestration-automation" target='_blank' rel='noreferrer noopener'>Case Management and Orchestration & Automation documentation</a>

* <a href="https://ibm.seismic.com/Link/Content/DCDH6waBAfRkeVnaQevXibAA" target='_blank' rel='noreferrer noopener'>IBM Security SOAR Client Presentation</a>

<br/>

##### Case management

Collaborate across teams with coordinated case management.

**Assets:**

*  <a href="https://www.ibm.com/docs/en/cloud-paks/cp-security/1.8?topic=team-introduction" target='_blank' rel='noreferrer noopener'>Case management documentation</a>

<br/>

##### Dynamic Playbook Designer

IBM Security SOAR helps make security alerts instantly actionable. With this SOAR solution, clients can make use of dynamic playbooks to provide analysts with a step‑by‑step guided response while improving collaboration and accelerating response times.

**Assets:**

* <a href="https://ibm.seismic.com/Link/Content/DCL4tBWew3Q0eCrxNlmZc33A" target='_blank' rel='noreferrer noopener'>Security Sales Talk - Cloud Pak for Security (CP4S) SOAR Playbook Designer Replay</a>

* <a href="https://www.ibm.com/docs/en/cloud-paks/cp-security/1.8?topic=guide-introduction#wp9000064" target='_blank' rel='noreferrer noopener'>Playbook Designer Guide</a>

<br/>

##### SOAR integrations

Automate repetitive tasks and reduce investigation time by integrating with other security tools.

**Assets:**

* <a href="https://ibm.seismic.com/Link/Content/DCDH6waBAfRkeVnaQevXibAA" target='_blank' rel='noreferrer noopener'>IBM Security SOAR Client Presentation</a>
