Cloud Pak for Data is IBMâ€™s integrated data and AI platform. With the ability to run anywhere and connect to data everywhere, it accelerates AI-powered transformation, unleashes productivity, and reduces complexity. As a single, unified platform, Cloud Pak for Data spans the entire data and analytics lifecycle with governance and industry-specific AI included from day 1.

There are three deployment options:

**System** - Provides a "cloud in a box" all-in-one hyper-converged infrastructure with the benefits of both on-premise and cloud deployment.

**Software** -  Provides the most comprehensive capabilities and ability to deploy anywhere and the best value for customers with strict requirements.

**As a Service** - Built on the Cloud and managed by IBM, this offers the fastest time-to-value and flexible pricing.
