## General product resources

Netezza was an appliance for advanced data warehouse and analytics platform. Now, IBM has reintroduced it as NPS or Netezza Performance server which is available on CP4D platform in three deployment options.

* NPS for CP4D aaS
* NPS for CP4D System
* NPS for CP4D

Review the IBM Netezza Performance Server <a href="https://www.ibm.com/products/netezza" target='_blank' rel='noreferrer noopener'>product page</a>.

<a href="https://www.ibm.com/docs/en/netezza" target='_blank' rel='noreferrer noopener'>Documentation</a>
