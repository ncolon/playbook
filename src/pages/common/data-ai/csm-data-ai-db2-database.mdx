## General product resources

IBM Db2 is a family of data management products, including the Db2 relational database. Db2 Database is available in:

* Cloud Pak for Data
* On Premises on Linux, Unix, Windows, z/OS
* On Cloud

Review the IBM Db2 Database <a href="https://www.ibm.com/products/db2-database" target='_blank' rel='noreferrer noopener'>product page</a>

<a href="https://www.ibm.com/docs/en/db2" target='_blank' rel='noreferrer noopener'>Documentation</a>
