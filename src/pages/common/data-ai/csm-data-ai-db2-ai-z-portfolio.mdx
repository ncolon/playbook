## General product resources


The core predictive capabilities in Db2 AI for z/OS (Db2ZAI) are built on top of IBM Watson® Machine Learning for z/OS, which enables real-time analytics with data in place on Db2 for z/OS.

IBM Cloud Pak for Data on z/OS helps to modernize Db2 applications.

Review the IBM Db2 AI Z portfolio <a href="http://www.redbooks.ibm.com/abstracts/redp5643.html" target='_blank' rel='noreferrer noopener'>Architectural overview, Installation & Configuration Red Book</a>.
