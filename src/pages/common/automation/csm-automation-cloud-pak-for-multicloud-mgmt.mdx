## ![use-case](../images/use-case-14.svg) Use Cases

IBM Cloud Pak for Multicloud Management provides governance and management capabilities required to implement your customer's use cases

Three entry points are implemented using one or more of the capabilities that are part of IBM Cloud Pak for Multicloud Management

#### Governance, risk, and compliance

Enforces policies and ensure compliance across hybrid infrastructures and applications.
Scans and monitors workloads continuously for vulnerabilities and mutations. Creates auditability of all policies, known vulnerability exposures and regulatory requirement tracking.

**Reference Architecture:**

<a href="https://www.ibm.com/cloud/architecture/architectures/sm-aiops" target='_blank' rel='noreferrer noopener'>AI for IT Operations architecture</a>

#### Cluster management

If you need cluster management capabilities for managing multiple clusters or, you need application management capabilities for Kubernetes-based application resources then an integration with Red Hat® Advanced Cluster Management for Kubernetes is a must

**Practice Leader Comments:** Cluster Management is achieved using Red Hat Advanced Cluster Management (RHACM). RHACM continues to be part of CP4MCM and customers can and should deploy it using their CP4MCM entitlement.

**Reference Architecture:**

<a href="https://www.ibm.com/cloud/architecture/architectures/sm-aiops" target='_blank' rel='noreferrer noopener'>AI for IT Operations architecture</a>

#### Infrastructure management

Infrastructure management delivers the insight, control, and automation enterprises need to address the challenges of managing virtual environments. This technology enables enterprises to improve visibility and control with virtual infrastructures.

**Practice Leader Comments:** Infrastructure management is achieved using CloudForms, CAM, and Terraform.

## Addressing roadmap questions on Cloud Pak for Multicloud Management

IBM has decided to merge Cloud Pak for Multicloud Management (CP4MCM) in to Cloud Pak for AIOps (CP4AIOPS) but has not announced End of Marketing or End of Support for CP4MCM yet. Please review the content in the links below to ensure that you understand the strategy and how to answer any questions your customer might have.
