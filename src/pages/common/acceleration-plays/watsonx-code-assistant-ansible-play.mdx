## Scope

**IBM watsonx Code Assistant for Red Hat Ansible Lightspeed** is part of a portfolio of purpose-built, generative AI-assisted products designed to accelerate development while maintaining trust, security, and compliance at the core. It demystifies the process of Ansible Playbook creation through generative AI-powered content recommendations. Purpose-built to accelerate IT Automation, this solution will deliver automation content recommendations that adhere to accepted best practices, for an enhanced Ansible experience users can be confident in. The solution can generate syntactically correct and contextually relevant content using natural language requests written in plain English text, helping to scale and expand access to automation across more of the business.

IBM watsonx Code Assistant for Red Hat Ansible Lightspeed employs a large language model that is trained on expansive datasets of Ansible Playbooks. The model can be tuned using an organizationâ€™s own data to understand the nuances of its automation syntax and structure. Users can accept suggestions as is or tweak the content recommendations to their exact needs, further improving time to value for automation and accelerating development cycles through AI content recommendations.
